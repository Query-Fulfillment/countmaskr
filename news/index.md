# Changelog

## countmaskr 0.1.1

CRAN release: 2026-04-10

### Initial release

- Initial CRAN release of
  [`countmaskr`](https://cran.r-project.org/web/packages/countmaskr/index.html).
- Provides automated small-cell suppression for one- and two-way
  frequency tables.
- Supports primary and secondary masking to help prevent indirect
  disclosure.
- Adds tools for vector- and table-level masking workflows, including
  [`mask_counts()`](https://query-fulfillment.github.io/countmaskr/reference/mask_counts.html),
  [`mask_counts_2()`](https://query-fulfillment.github.io/countmaskr/reference/mask_counts_2.html),
  [`perturb_counts()`](https://query-fulfillment.github.io/countmaskr/reference/perturb_counts.html)
  and
  [`mask_table()`](https://query-fulfillment.github.io/countmaskr/reference/mask_table.html).
- Includes the example dataset
  [`countmaskr_data`](https://query-fulfillment.github.io/countmaskr/reference/countmaskr_data.html).
