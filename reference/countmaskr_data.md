# countmaskr dataset

A synthetic dataset of 1500 subjects with demographic information
ascertaining at least one category from the demographics has counts less
than 11.

## Usage

``` r
data(countmaskr_data)
```

## Format

### `countmaskr_data`

A data frame with 1500 rows and 6 columns:

- id:

  Subject id

- age:

  Age in years as integer

- gender:

  Administrative sex

- race:

  Race

- ethnicity:

  Ethnicity

- age_group:

  Age as categorical variable
